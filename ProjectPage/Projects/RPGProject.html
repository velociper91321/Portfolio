Modular Top-Down RPG With Unreal Engine's Gameplay Ability System

Modularity of an RPG System

An easily customizeable system with the help of GAS
This modular RPG project is built with Unreal Engine's Gameplay Ability System, offering a flexible framework for creating customizable abilities, character systems, and gameplay mechanics. Designed to be highly extensible, it allows developers to easily implement and iterate on RPG features like skills, stats, status effects, and combat interactions, making it ideal for crafting unique and dynamic gameplay experiences.
